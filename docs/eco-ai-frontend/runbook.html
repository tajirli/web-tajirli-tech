Local Setup

  1. npm install
  2. Set VITE_API_URL for the backend host
  3. npm run dev
  4. Verify language switch and portfolio carousel
  5. Submit a contact form against a live backend

Checks Before Release

  • Confirm images and logo assets load correctly.
  • Validate FR/EN/AR rendering and RTL behavior.
  • Test the contact form success and error states.
  • Verify analytics and external links remain intact.
