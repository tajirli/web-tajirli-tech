Local Setup

  1. npm install
  2. npm start
  3. npm run android or npm run ios
  4. Confirm API config resolves active backend URL
  5. Login and verify data stores hydrate

Quality Checks

  • npm run type-check
  • npm test
  • Smoke test products/orders/settings flows
  • Validate auth pending-approval behavior
  • Validate app foreground/background sync behavior

Release Checklist

Step Action Expected result
1 Update version and release notes Release metadata aligned with code
2 Run type-check and tests No blocking issues
3 Validate high-risk screens and auth gate Navigation and permissions are correct
4 Validate sync + realtime behavior No duplicate polling or stale state
5 Push only to approved origin repository No accidental push to legacy remote

Operations baseline maintained in this document.
