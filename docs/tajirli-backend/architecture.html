Layer Responsibilities

  • Entry initializes HTTP server, database, Socket.IO, and startup jobs.
  • Core layer enforces security headers, rate limiting, and request logs.
  • Route layer separates public and protected endpoint families.
  • Feature layer encapsulates domain behavior and HTTP handlers.
  • Service layer hosts integrations and reusable adapters.

Architecture Risks

  • CORS origin drift can block valid clients during rollout.
  • Mixed legacy and modular paths require import discipline.
  • Webhook verification requires raw body path ordering to remain stable.
  • Realtime rooms must stay aligned with frontend auth tokens.
