Primary Modules

Module Key Files Main Responsibility Notes
Routing + Shell src/App.jsx Route wiring, lazy loading, chrome visibility, pageview tracking Admin and storefront routes are split by guard + path prefix
Product Domain src/features/products/services/ProductService.ts Product listing, category filter, typed product fetching Exists with legacy getProducts in src/api.js
Order Domain src/features/orders/services/OrderService.ts Create order, delivery search, delivery pricing fetch Request payload adds source/platform = website
Settings + Branding src/context/SiteSettingsContext.jsx Manage admin-customizable fields and push settings to backend Also injects/removes tracking pixels dynamically
Security Utilities src/utils/adminAuth.js Admin login, role check, session persistence, current-user fetch Token is stored in localStorage
Tracking Utilities src/utils/metaPixel.js, src/utils/useVisitorTracker.js Marketing and behavior events Visitor tracker disabled on /admin paths
SEO/OG Function api/products/[slug].js Bot-aware OG pre-render for product sharing Serves dist/index.html for non-bot traffic
