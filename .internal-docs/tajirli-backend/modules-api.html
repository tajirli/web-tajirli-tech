Core Modules

Module Main Files Responsibility Status

Operational API Notes

Concern Current Behavior Owner Surface
Authentication JWT middleware protects business routes, auth routes remain public/self-scoped src/core/middleware/auth + src/routes/auth.js
Rate Limiting Default limiter applied on /api before endpoint dispatch src/core/security/rateLimiter.js

Realtime Sync Socket.IO rooms handle dashboard, conversation, and sync broadcasts src/server.js socket handlers
