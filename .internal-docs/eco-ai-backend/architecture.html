Layer Responsibilities

  • Entry: server.js binds to 0.0.0.0 and reads environment variables.
  • Data: Mongoose handles User and Contact persistence.
  • Auth: JWT token creation on register and login.
  • Delivery: Brevo sends email after contact submission.

Architecture Risks

  • Hard-coded allowed origins need environment control in production.
  • Contact endpoint depends on Brevo API availability.
  • JWT secret must be set outside local fallback environments.
  • Model schema is minimal and should be extended for production entitlements.
