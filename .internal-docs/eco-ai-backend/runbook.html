Local Setup

  1. npm install
  2. Create .env with MONGODB_URI, JWT_SECRET, BREVO_API_KEY
  3. node server.js
  4. Check GET / and POST /contact
  5. Validate auth routes with test user data

Production Checks

  • Confirm allowed origins match deployed frontend URL.
  • Verify MongoDB connection and write permissions.
  • Test Brevo email delivery from /contact.
  • Confirm JWT token issuance on register/login.
