Frontend Responsibilities

  • Present the public sales and onboarding story.
  • Collect leads with client-side validation.
  • Translate the UI for three supported languages.
  • Keep SEO-friendly, static-friendly markup.

Backend Contract

  • Uses `VITE_API_URL` for environment-specific API access.
  • Posts contact payloads to `POST /contact`.
  • Expects JSON response from the backend for form submission success.
  • Depends on MongoDB and Brevo handling server side.
